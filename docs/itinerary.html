itinerary¶

itinerary.from_excel(project, filename)[source]¶

Read itineraries from an excel file. Excel header: Vehicle ID, Start time (hour), End time (hour), Distance (mi), P_max (W), Location, NHTS HH Wt.

Parameters:
  • project (Project) – empty project
  • filename (string) – relative or absolute path to the excel file
Returns:

project – project assigned with vehicles
Return type:

Project
itinerary.unique_location_category(location_category, project)[source]¶

Compare input location category with locations in the project, return a new location and update the project or just return an existing location.

Parameters:
  • location_category (string) – location category (Home, Work, ...)
  • project (Project) – current project
Returns:

location – either a new or an existing location
Return type:

Location
